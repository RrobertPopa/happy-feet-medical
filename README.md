# Happy Feet Medical — website demo

Demo de prezentare pentru **Happy Feet Medical** (pedichiură cu notă medicală, București).
Site static, self-contained (HTML + CSS + JS inline, asseturi locale, căi relative — merge și deschis direct de pe disc).

## Deploy pe Vercel
1. Import acest repo în Vercel → Framework Preset: **Other** (fără build).
2. Output: rădăcina repo-ului (`index.html`).
3. Domeniu temporar `*.vercel.app` sau domeniu propriu la closing.

## De completat înainte de live

- Recenziile reale (acum 3 mostre ilustrative marcate `AGENT:`).
